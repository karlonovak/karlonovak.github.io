About Myself

My name is Karlo Novak and I'm a young and extremely ambitious software developer based in Zagreb, Croatia.

Currently I work at Greyp Bikes (Rimac Automobili) writing and architecting huge and distributed IoT systems for world's most powerful electric bikes.

My main point of interest is JVM and its ecosystem and I specialize in writing Java based applications mostly with Spring framework (Core, Web, Data, Security, Boot, Remoting...) although I also feel comfortable with writing in Scala as my hobbyist language. So far I've been using lots of different technologies, both on the frontend and backend such as JAX-RS, JAX-WS, raw Servlets, Apache Tomcat, Websphere Application Server, AngularJS, PrimeFaces... but among all, Spring stays my greatest love.

Besides JVM, I never hesitate to cope with any other programming language or technology so there is a huge list of things I've tried or have been working with but it would take too much place if I started writing.
